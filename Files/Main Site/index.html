Service

Parking Management system, the motive of our project is to make parking, a huge hassle in our country, much more simpler and easier.
This Python programme will analyse the empty and occupied spots, make calculations by itself based on the user's request and assign an empty spot, if available, within fractions of a second which will make parking more accurate, fast and organized.

Software Requirement

Python 3.9.6 IDLE,
PyCharm (Community/Pro Edition), Required Python Modules,
Google Chrome,
Windows 10 or Windows 11,
Active Internet Connection

Hardware Requirement

Processor: 1 gigahertz (GHz),
RAM: 4 GB, Hard Disk:20 GB, Graphics Card: DirectX 9 or later with WDDM 1.0 driver,
Display: 800 x 600,
Input Devices : Mouse and Keyboard

Advantages

The parking management system isnt relayed on humans to process the information and manage the available parking spaces, this means that the system is more likely to make fewer errors and make the experience more smooth.

Development

To make the user experience much more seamless, we can add cameras to the system that can be used for automatic license plate recognition. This can cut down on errors and make paying for the parking service more smooth.
